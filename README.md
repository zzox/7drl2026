`sons` is a challenging grid-based autobattler with genetic party mechanics. You fight through a family tree while genetically altering your party to create the best possible battler.

This game uses the [kha](https://kha.tech) multimedia framework.  More framework specific code lives in `src/core/` while `src/game/` has game code.
